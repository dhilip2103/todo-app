# 📖 Overview

Todo-App is a modern cross-platform task management application developed using Flutter and Firebase.

It enables users to securely register, log in, and efficiently manage their daily tasks with real-time cloud synchronization.

This project demonstrates Flutter application development, Firebase Authentication, Cloud Firestore integration, CRUD operations, responsive UI design, and clean project architecture.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/dhilip2103/
```

Go inside project

```bash
cd todo-app
```

Install packages

```bash
flutter pub get
```

Configure Firebase

- Enable Firebase Authentication
- Enable Cloud Firestore
- Download Firebase Configuration

Run the application

```bash
flutter run
```
